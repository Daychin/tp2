Pour les pseudo-classes, j' ai compris ceci: la syntaxe et les effets. Le syntaxe est simple. Il faut avoir cette structure: sélecteur:pseudo-class{propriétée: valeur}. Par exemple, vous voulais les liens change de couleur (rouge) quand vous flotter, alors le code est ceci: a:hover {color: red}. Ensuite, il y a plusieurs effets comme le hover ( qui change le couleur du sélecteur ), le visited ( change de couleur apres avoir clicker sur le lien) etc… Pour l opacité, j' ai compris la même chose. La syntaxe est comme ceci: img{ opacity: opacité que vous voulez }. L' effet est assez logique, l' image devient plus brumeuse. De plus, on peut ajouter des pseudo-classes comme le hover.

Pour les galeries d image, la syntaxe est différente des autres. Il faut ajouter plusieurs code de css a faire. Pour commencer, il faut que l' image soit le texte soit dans un div. Ensuite, on peut mettre un lien pour l' image et l' ajouter. De plus, il faut créer une autre div dans le div pour écrire une petite description. Dans css, il faut ajouter un border et autre truc pour les ajuster à votre préférence. Les bordures des images est très simple. Comme le nom le mentionne, il ajuste le bord de l' image. Le code est comme ceci: border-radius: X%.
